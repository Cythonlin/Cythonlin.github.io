AI => Tensorflow2.0（stable）降临

前言

就在今天凌晨，TF20标准版，降临，可喜可贺。
因此，我认为，值得记录。。。

Github URL

https://github.com/tensorflow/tensorflow/releases/tag/v2.0.0

"goldiegadde released this 19 hours ago" ~~ yeah

参考Beta环境

我之前写过一篇2.0-GPU Beta环境， 你若连Tensorflow都每装过，那么移步：
传送门：https://segmentfault.com/a/1190000020304638

更新至TF20-GPU命令

pip install -U tensorflow-gpu -i https://pypi.douban.com/simple

查看版本

>>> import tensorflow as tf       # 运行后，会输出一段关于CUDA的 INFO
>>> tf.__version__
'2.0.0'

Pycharm语法高亮问题

Pycharm当前最高可用版是 v2版本。
但是TF20(Stable)是新出的， 所以Pycharm中是没有TF20语法高亮的。

其实你可以等等。Pychram等更新到 v3版本（11月份左右， Pycharm出标准版）

如果你等不及了，没问题。 给你个 v3预先版地址，自己下载。 绝对支持 TF20语法高亮，亲测！
Pycharm3.0+传送门：https://www.jetbrains.com/pycharm/nextversion/

强调导入问题

TF2O 的 Keras 是这样导入的： （因为每次更新TF，Adam很容易找不到，所以以Adam导入为例）

方式1：
import tensorflow as tf
import tensorflow.keras as tk
tk.optimizers.Adam()
方式2：
import tensorflow as tf
from tensorflow.keras import optimizers
optimizers.Adam()
语法高亮这个问题是我去Stack找的：
原标准回答传送门： https://stackoverflow.com/questions/58188704/unable-to-import-kerasfrom-tensorflow-2-0-in-pycharm/
