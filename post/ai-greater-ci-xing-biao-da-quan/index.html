GO => 谈一谈Goland/Pycharm/Idea全局搜索为何偶尔失灵的问题

2020-09-29

众所周知

JetBrains全家桶系列默认全局搜索快捷键为:
ctrl + shift + f

为何上述快捷键偶尔失灵?

不想看原因的可以直接跳到最后看2种解决办法

微软拼音切换简体/繁体：

一直使用的是Win10自带的微软拼音（其他输入法没用过，应该类似）
ctrl + shift + f
这里切换简体/繁体有个条件：
只有你当前输入法状态为 “中文输入法”时， 切换简/繁（ctrl+shift+f）快捷键才会生效
如果当前输入法状态为 “英文输入法”时， 切换简/繁（ctrl+shift+f）快捷键 不起作用。
到这里应该可以想出pycharm/goland/...全局搜索为何失灵了吧~~

失灵原因

因为写代码时，输入法可能会 中/英 切换。
当前我们的输入法状态为 “中文” 时： ctrl+shift+f 的作用是切换 繁/简 输入法
当前我们的输入法状态为 “英文” 时： ctrl+shift+f 的作用就是我们想要的《全局搜索》

解决办法

解决办法1（切换习惯）

根据失灵原因，我们想要使用JetBrains系列产品全局搜索时，我们可以注意自己的输入法是否是"英文状态"，若不是，则切换到英文状态：
shift # 微软拼音切换 中/英 是 shift ，其他输入法不清楚
对我而言，这是习惯问题，不存在输入法冲突，如果没切换中/英这个习惯，可以按照如下改快捷键

解决办法2（改快捷键）

如果执意要改快捷键，可以改快捷键 (因为我已经习惯了中/英输入法切换方式)，所以我是不改的。
按照如下配置流程走：
File ->
Settings ->
Keymap ->
在搜索快捷键的输入框输入： find in path
双击 "find in path" ->
Add Keyboard Shortcut（然后键盘按下你像设置的快捷键）
（我个人推荐是 ctrl+shift+|），因为这个快捷键未被占用 ~
ok ->
ok
